# Lab6 GUI Project

This project is a simple **Login GUI** made using Java Swing.  

## Screens
- **Login Screen** – includes username and password fields, and `Login` and `Register` buttons.

## Features
- Clicking buttons prints information to the console:  
  - `Login` button → shows the username in the console  
  - `Register` button → prints that the register button was clicked  

## Files
- `LoginGUI.java` – main program code  
- `docs/` – folder containing GUI screenshots and extra information  
- `README.md` – this description
